# Arc Pad. A cheap, open hardware and easy to make Macro Pad.

<img width="1920" height="1080" alt="bottom" src="https://github.com/user-attachments/assets/a313379b-d3a6-4110-a913-1128a9d24fe1" />



# Only 28 dollars for all parts.
This is the cost for all part for one Arc Pad. Of course price will be different at times. For me the price included one pcb, one 0.96 Oled, TFT SPI 1.8 INCH, 9 XM SWITCHES, 9 KEYCAPS , TWO 6MM TACTILE BUTTONS AND ONE RASPBERRY PI PICO 2 W.

<img width="1920" height="1080" alt="backside" src="https://github.com/user-attachments/assets/576b9bb4-d543-4035-8d75-b1b28ccbb72f" />



# Powered by a Raspberry Pi Pico 2 W.

It uses a Raspberry Pi Pico 2 W for a powerful and reliable use. Feel free to change the microcontroller to you needs.
